Ikuti Langkah-Langkah Berikut!

  • STEP 1

    Persiapkan landasan api

  • STEP 2

    Buat tumpukan ranting-ranting kecil kering dan daun-daun kering serta bahan pematik api lainnya di atas landasan api

    Mulai nyalakan api dengan membakar bahan pematik api seperti kawul, jamur api, ranting dan daun kering

    Kenali tanda api menyala yaitu dengan adanya asap

  • STEP 5

    Memberikan tiupan udara atau kipas dengan kencang sumber awal api agar api semakin membesar serta tambahkan bahan pematik api secara bertahap ketika api mulai menyala
